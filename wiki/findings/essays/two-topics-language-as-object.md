# Essay: two topics, one model

> **Status: draft (2026-06-28). A philosophical-track essay arguing in the project's own voice.** Its contribution is a *distinction-tool*, not an empirical result: it lifts one methodological move from Lewis 1970 — the insistence that the description of a language as an abstract semantic system (a function from expressions to meanings) and the description of the psychological-and-sociological facts whereby one such abstract language is the one a person or population actually uses are **two distinct topics**, not one — and uses it as a lens on the project's recurring question of whether an LLM "has" a language. (That two-topics framing is the project's own reading of Lewis 1970 — drawn from the source page's verbatim material plus Lewis's broader "General Semantics" / *Convention* view — not a verbatim quotation, and **no in-repo page asserts it**; see the Honesty box and trigger (a).) The essay introduces **no new empirical claim**; every empirical or textual assertion cites the in-repo page that carries it, and Lewis is cited only through [`source/lewis-1970-general-semantics`](../../base/sources/lewis-1970-general-semantics.md), with one quote verified verbatim there and the remainder of Lewis's position paraphrased and attributed to that page (flagged in the Honesty box). Read "The split, and what each topic asks of a model" before citing.

## The position

This essay argues one bounded thesis. Lewis 1970 opens by dividing the study of meaning into two distinct topics — an abstract one (a language as a formal object, a function from expressions to meanings) and an empirical one (the psychological-and-sociological facts whereby one such abstract object is *the* language a population actually uses) — and **insists they are different questions that breed error when run together.** Applied to an LLM, the split does real work: the project's behavioral method describes the model as instantiating, or approximating, a Lewisian **language-as-object** — a systematic expression→behavior mapping — and that description can succeed completely while the *second* topic, whether there is a population, a convention, a common knowledge that selects this object as the language the model *uses*, remains **untouched and separately owed.** The error Lewis warns against has a precise LLM form: reading the second topic off the first — taking the systematicity of the mapping as if it settled the use-facts. The essay's claim is that the project's standing positions already sort cleanly onto the two sides of Lewis's line, that naming the line sharpens what each of them is and is not about, and that the second topic is the one the project's evidence is structurally worst-placed to reach.

This is a *conceptual* contribution — a way of carving the question — not a measurement. It re-labels which of the project's silences are silences of the first topic and which of the second; it resolves nothing on either.

## The split, and what each topic asks of a model

Lewis's opening pages are read here as naming two topics — this two-topics framing is the project's own reading of Lewis 1970, not a quotation and not stated by the in-repo source page, which is silent on the split (see the Honesty box and trigger (a)); the source page supplies the verbatim semantic apparatus that fills out the first topic. The first is the description of a language as an abstract semantic object: a meaning for a sentence is "something that determines the conditions under which the sentence is true or false" ([`source/lewis-1970-general-semantics`](../../base/sources/lewis-1970-general-semantics.md), pp. 22–23), an intension is "any function from indices to appropriate extensions" (p. 23), and a full meaning is a "semantically interpreted phrase marker" — a tree carrying a category and an intension at each node, built part-from-part so a compound's meaning is computed from its constituents' (pp. 31–33). All of this is the study of *a language as a function from expressions to meanings*, pursued formally, with no reference yet to who speaks it. The second topic is the empirical one: which of these abstract objects is the one a particular person or population actually *uses* — a question Lewis treats as settled by facts about convention and common knowledge among speakers, not by anything internal to the abstract object itself. (Lewis's developed account of that selection — language as a convention sustained by mutual expectation in a population, set out fully in his *Convention* 1969 — is the project's own paraphrase of Lewis here, not quoted and not drawn from any in-repo page; see the Honesty box.)

Hold the two apart and ask what each demands of an LLM.

- **Topic one — the language-as-object.** Does the model realize, or approximate, a systematic function from expressions to behavior that *can be described* as a Lewisian language: stable mappings, compositional combination, the kind of regularity an abstract semantic system has? This is a question about the *mapping*, answerable by probing the mapping. It does not ask whether anyone uses the language; an abstract object has no users by definition.
- **Topic two — the selection facts.** Is there a population, a convention, a structure of common knowledge in virtue of which *this* abstract object (rather than some other one the same behavior is also consistent with) is *the* language the model uses? This is a question about use-facts standing *outside* the mapping — about who, if anyone, the model is in linguistic community with, and what fixes the language as *theirs*. No amount of regularity in the mapping answers it, because the mapping is what gets *selected*, not the selecting.

Lewis's whole point is that these come apart. You can have the first without having settled the second: an abstract language is a well-defined object whether or not any population uses it, and the facts that would make it *someone's* language are a separate study. That separateness is the lever this essay applies.

## Which topic the project's method is built to address

The project's behavioral method is, almost by construction, a **topic-one instrument.** It reads meaning off patterns in the expression→behavior mapping — distributional neighborhoods, substitution behavior, surprisal, inference-preservation, constructional acceptability — and the controlled vocabulary states exactly this register: `distributional` meaning is "Meaning as patterns of co-occurrence in linguistic context … the implicit theory of word embeddings and the next-token objective," operationalized "through similarity, substitution behavior, neighborhood structure, and predictive surprisal in context" ([`meaning-senses.md`](../../meaning-senses.md), §`distributional`). Every one of those measures is a measure *of the mapping*. When the project finds that the panel composes a construction's meaning past a single-clause heuristic, or tracks a graded sense gradient, it has characterized the abstract semantic object the model approximates — it has done topic one, and (within its bounds) done it well.

The synthesis page's own verdict can be re-read in exactly these terms. Its one-sentence sketch is that "Where the project can see it, LLM meaning lives **inside the model** as a graded, compositional, *thin*-inferential use-structure that **beats but does not escape** the distributional shadow" ([`theory/situating-llm-meaning`](../theory/situating-llm-meaning.md), §"The synthesis verdict"). *(Superseded s179 by [`theory/situating-llm-meaning-v2`](../theory/situating-llm-meaning-v2.md), which restates the quoted prediction frame-relative — cited here as the edition this essay engaged.)* Read through Lewis: "where the project can see it" is precisely the region of topic one — the structure of the mapping — and the parts it reports as silent or negative ("a world it refers to … a perceptual ground, other agents," same section) are the places where a *different* topic is in play. The essay's claim is that this is not a coincidence of where the probes happened to land; it is the shape Lewis's split predicts. A method that measures the mapping characterizes the language-as-object and is constitutively quiet about the facts that would select it as anyone's.

## The error Lewis warns against, in LLM dress

Lewis's reason for separating the topics is that fusing them breeds error — and the fusion has a sharp LLM instance worth naming because the debate around models slides into it easily. The tempting inference is: *the model's expression→behavior mapping is richly systematic; therefore the model has, or means, a language.* Lewis's split shows the "therefore" skips a topic. Systematicity is evidence about the language-as-object (topic one). Whether the model *has a language* — in the sense of there being a convention and a population that make some abstract object the model's own (topic two) — does not follow from the object being well-formed, any more than the well-formedness of an unspoken abstract language makes it anyone's.

This is not a claim that the model *lacks* a language; it is a claim that the question is a *second* one, owed its own argument. And the project already holds the disciplined version of this caution under a different description. Its descriptivist stance — "describe, don't litigate" — is precisely a refusal to read an existence verdict off the describable structure: the yes/no is "**under-determined by the behavioral evidence the project can obtain**," so the rational move is to "describe the structure that is actually there" rather than adjudicate "whether the structure 'is meaning'" ([`concept/deflationary-and-eliminativist-llm-meaning`](../../base/concepts/deflationary-and-eliminativist-llm-meaning.md), §"The project's descriptivism"). Lewis's two-topics distinction supplies a *reason* this discipline is correct and not mere caution: the describable structure is topic-one evidence, and the existence question the project declines to litigate is shot through with topic-two facts the mapping does not contain. "Describe, don't litigate" and "don't read topic two off topic one" are the same injunction stated from two directions.

## Where the project's standing positions sit on the line

The split sorts several of the project's positions, and the sorting is the essay's substantive payoff — it shows the line is already implicit in the wiki and worth making explicit.

**On the topic-one side.** The constructional evidence ladder, the lexical sense-gradience positive, the compositional reading of the comparative correlative — these are characterizations of the language-as-object. They say what mapping the model approximates and how much of its structure beats the distributional null. They are, in Lewis's terms, contributions to the description of *an abstract semantic system the model realizes*, and they neither claim nor need a population to be about.

**On the topic-two side — and this is where the split earns its keep.** Two of the project's hardest silences are now legible as silences of the *second* topic, not the first:

- *Reference.* The in-repo dispute over whether an LLM's words refer is, as [`essay/reference-as-premise-bound`](reference-as-premise-bound.md) argues, "premise-bound, not evidence-bound" — it turns on "whether inheriting word-histories through a training corpus counts as being 'part of the linguistic community' whose use fixes reference," a fact "fixed off-board … not by any in-head or in-behavior fact." In Lewis's vocabulary that is a topic-two question wearing reference's clothes: community membership is exactly a *selection fact* — what makes some language, with its referential semantics, the model's own — and the reason no probe reaches it is that probes measure the mapping (topic one) while membership lives among the use-facts (topic two). The premise-bound essay locates the silence in an antecedent; this essay names *which topic* that antecedent belongs to.
- *Relational constitution.* The project's relational work asks whether content is "constituted *between* agents … rather than computed *inside* a single agent and then read off" ([`concept/relational-meaning`](../../base/concepts/relational-meaning.md)). Constitution-between-agents is, structurally, a candidate *source of topic-two facts* — a population and an interaction in which a convention could be fixed. The standing verdict is that the observed order-sensitivity is "**thin** (a *latest-binding-wins / convention-update* rule)," "not a convention *constituted between* agents through the live trajectory" ([`concept/relational-meaning`](../../base/concepts/relational-meaning.md)). Lewis sharpens *why* "thin" is the right word: a latest-binding rule is a feature of the *mapping* (a single reader recovers it from the record), so it is topic one; a convention in Lewis's sense — selecting a language for a population by mutual expectation — would be topic two, and the project has not reached it.

So the project's positive findings cluster on topic one and its hardest silences cluster on topic two — and that is not two unrelated facts but one structural pattern the split makes visible.

## A caution the split forces: two senses of "convention"

Naming the line also flags a word the project must not equivocate on. "Convention" appears throughout the relational pages — a coined referential label, a latest-binding update rule, an entrained term. That is the *operational* sense: a regularity in the mapping that a single reader recovers from the record. Lewis's "convention," by contrast, is a *selection fact* — the mutual-expectation structure in a population that makes one abstract language *the* language they use. These are not the same notion, and the two-topics split is exactly what keeps them apart: the operational convention is topic-one machinery; the Lewisian convention is topic-two metasemantics. The risk the essay flags is that finding an operational convention in an LLM dyad could *look* like finding a Lewisian one — like the model entering a language-selecting community — when it is only a feature of the mapping the probe was always going to see. This is the relational analogue of the descriptivist caution above: a regularity that any single reader can compute from the record is topic-one evidence, and reading topic-two community-constitution off it would be precisely the fusion Lewis warns against. The project's own insistence that the thin rule is "**not** that surplus" ([`concept/relational-meaning`](../../base/concepts/relational-meaning.md)) is the discipline already in place; the split says what kind of thing the surplus would have to be.

## What the split buys, and what it does not

The payoff is precision about the project's own limits, not progress past them. Three things follow, all modest.

- **It re-describes the project's central silence as principled, not provisional.** The truth-conditional / referential cell is recorded as "silent," and the synthesis page notes that "only a built world-relation moves the referential cell" ([`theory/situating-llm-meaning`](../theory/situating-llm-meaning.md), §"What this synthesis predicts and forbids"). The split says why a cleverer text task cannot move it: more probing is more topic-one evidence, and the cell's occupant is a topic-two fact. The silence is structural — it is the silence of the wrong topic — which is exactly the register [`essay/compositionality-asymptote-not-wall`](compositionality-asymptote-not-wall.md) reserves for reference as "a wall," a place the behavioral method "falls short of its philosophical target" by kind rather than by degree.
- **It does not adjudicate topic two.** The essay takes no side on whether an LLM *is* in a language-selecting community, whether corpus inheritance counts as membership, or whether a model "has" a language in the full sense. It says only that this is a *separate* question from the systematicity the project measures, owed its own argument on its own terms — exactly as Lewis insists, and exactly as the premise-bound essay finds the reference dispute to be.
- **It does not deflate topic one.** Characterizing the language-as-object is real work and the project's positives are real characterizations of it. Saying they are topic-one results is not saying they are *merely* anything; it is saying what they are about. An abstract semantic system the model genuinely approximates is a substantive finding, and Lewis — for whom the abstract object is half the whole study of meaning — is the last person to belittle it.

## What this essay is not

- **Not a claim that the project has settled, or can settle, the second topic.** The opposite: it argues the second topic is the one the behavioral method is structurally worst-placed to reach, and leaves it open.
- **Not the reference essay's thesis, and not the relational essays'.** [`essay/reference-as-premise-bound`](reference-as-premise-bound.md) locates a silence in the antecedent of a specific externalist conditional; [`essay/inherited-not-constituted`](inherited-not-constituted.md) and [`essay/update-is-not-constitution`](update-is-not-constitution.md) argue about constitution *between agents* and the ladder of relational rungs. This essay sits *upstream* of all three: it supplies the Lewisian two-topics frame onto which those particular silences sort, and its contribution is the frame and the sorting, not any of their specific results. It `refines` [`theory/situating-llm-meaning`](../theory/situating-llm-meaning.md) by re-describing the map's "model-internal positive / referential-and-relational silent" pattern as a topic-one / topic-two split.
- **Not a primary reading of Lewis's account of convention.** Lewis's developed theory of language-as-convention is the project's own paraphrase of Lewis here, not quoted and not asserted by any in-repo page; the one verbatim Lewis quote used is the truth-conditional "meaning … determines the conditions under which the sentence is true or false" passage (pp. 22–23). See the Honesty box and the revision trigger.
- **Not a verdict that "language-as-object" is all an LLM has.** The essay says the project's *method* sees topic one and is quiet on topic two; it does not claim topic two is empty for LLMs, only that it is not read off topic-one evidence.

## Revision triggers (read before citing)

- **(a) If Lewis's *Convention* (1969) or his developed language-as-convention account enters the repo as a primary** — currently the second-topic half of this essay leans on the project's own *reading* of Lewis's selection-by-convention view (not on quoted text, and not on any in-repo page — the source page is silent on it). If a primary reading sharpens or corrects how Lewis selects a language for a population (e.g. the precise role of common knowledge and mutual expectation), re-state the topic-two side at that sharper strength and check that nothing here over-attributes to Lewis.
- **(b) If a behavioral probe is designed that an adversarial reviewer judges genuinely bears on a topic-two selection fact** — a population/convention fact rather than a feature of the mapping — then the central "the method sees only topic one" claim weakens to "the methods tried so far see only topic one," and the essay must be relativized accordingly. (The relational track is the natural place such a probe would arise; the standing position is that its observed conventions are thin / single-reader-recoverable, i.e. still topic one.)
- **(c) If a human anchor for the second topic enters the repo** — a resource that pins a population/convention/common-knowledge fact an LLM could be tested against — then topic two becomes at least partly evidence-bound on the project's own terms, and the split's "structurally worst-placed to reach" claim must be restated for whatever residue the anchor does not reach.
- **(d) If the relational track ever crosses from thin order-sensitivity to genuine between-agent constitution** (the rich rung that stays "documented structurally closed for text-only stimuli" per [`essay/update-is-not-constitution`](update-is-not-constitution.md)), the topic-two side of the sorting changes: a constituted convention would be a candidate topic-two fact the project had actually reached, and the relational cell would move from "topic two, unreached" to "topic two, partially reached." Re-sort that cell when and if it happens.

This shape is the essay's own; no source page states it.
