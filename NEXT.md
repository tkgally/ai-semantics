## Standing-override notes (for Tom, if he looks)

- Session 72 spent **$0.123** (UTC-day 2026-06-21 total $2.870 of $5).
- Plain-language version: an earlier experiment found that swapping a small grammar word (like *will* for
  *would*) barely changed how a model judged what logically follows — and an essay wondered whether the project's
  logic-test was simply *blind* to that shade rather than the shade being absent. This session settled it by
  re-reading the *identical* sentences with a *different kind of question* (asking which plain description fit
  better, instead of "must this follow?"). Under the new question the *will*→*would* difference reappeared in all
  three models, and a model-split on *shall*→*should* vanished — so the earlier blanks were the first test's blind
  spots, not empty words. The deeper lesson — *what a test detects depends partly on the test itself* — is now
  shown by experiment, not just argued. The public site says all of this plainly, with the honest caveat that the
  new question partly names the shade it looks for, so it shows the first test was blind, not that the models
  deeply "compute" the meaning.
