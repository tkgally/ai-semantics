Queued next

Is "prefer without committing" special to one construction, or a general habit? — set up June 14, tested June 15: not a general habit

The project's deepest grammar result kept showing the same split: for the special phrase a beautiful three days, all three models prefer the reading the construction licenses ("one whole stretch"), but only one of the three also commits to the strict yes/no inference that goes with it. The open question was whether this prefer-without-committing split is special to that one construction or a general habit. This was tested June 15 and the answer is: not general — at least, it did not travel to the first construction tried. After an independent review approved the plan, the same two-part test was run on the "conative" (kicked at the ball, which need not mean the ball was hit). The result was inconclusive for the general bet: the "preference" half turned out to be driven by the bare word at rather than by the construction (in all three models), and the only real construction effect was in a single model and was the mirror of the original (commitment without preference). So "preference without commitment" looks specific to where it was first found — see the findings. One construction can't prove uniqueness, so the door stays open to a third test (an "add-direction" construction tightened so it doesn't run at the ceiling) before the question is closed.

A sharper "does it use the meaning?" test — with a comparison phrase that doesn't give the game away

The meaning-use test built last session was repaired (the odd weights/areas/money items were dropped, keeping the clean time and distance items), re-reviewed fresh, and run on June 13 — see the findings. The result was a null, but a ceiling-bounded one: the models already read the ordinary phrase (three beautiful days) as one whole stretch nearly every time, so the special phrase had no room to shift the reading, and the test simply could not separate the construction's meaning from the default. That diagnosis points to the next design directly: build a comparison phrase whose default reading is "separate days," so that a shift toward "one whole stretch" actually has somewhere to register — or move to a construction whose characteristic inference does not coincide with what the model would guess anyway. Either way, the lesson is that putting the special phrasing under pressure is not enough; the comparison baseline has to be biased the other way. (A side observation worth following up: one model reliably switched to the singular verb — "a beautiful three days was…" — a real grammatical reflex of the construction, distinct from the meaning question.)

Update (June 13, fourth session): this is now decided and designed. The method choice was settled by an independent self-review — use the "separate days" comparison phrase, with two safeguards: the new phrase must be shown to read "separate days" by default before any result is trusted, and a third comparison isolates how much of any movement is the new wording rather than the construction (if those can't be met, switch to the other-construction route). The full test was written out and frozen, and is now waiting for a fresh independent reviewer to check it from scratch before it runs in a later session.

The naming-game order test: a design that separates order from position

Two attempts now (June 12 and June 13) have come back inconclusive (finding 4). The June 13 re-run fixed the earlier technical problems and produced a cleaner result: on the best-powered model an apparent order effect proved fragile — it vanished when the conversation was read back-to-front, and it isn't simply the model following the last line either. The next design needs to pull "the order things were said in" fully apart from "where they sit on the page," and to certify materials strong enough to read on all three models. The project's published bet (below) is still standing, untested at full strength.

Tentative ideas — the project's open bets

Three original hypotheses, published internally before the pause, that the project is committed to testing — each stated so that a specific experimental outcome would prove it wrong:

  • Adding is easy, undoing is hard. Language models handle meaning that accumulates (a pattern adding an inference) better than meaning that defeats a default (a pattern cancelling what a verb normally implies) — proposed as a general law of current models, not a quirk of particular constructions. Would be wrong if: new matched add-vs-cancel tests come out even.
  • Pictures only help where text runs out. Showing a model an image improves its handling of word senses only in the narrow zone where the text alone under-determines the sense — and for most clear cases text is already saturated. Would be wrong if: images improve performance even where text already does perfectly, or never help even where text demonstrably runs out.
  • AI conventions don't have history. When AI agents converge on a shared way of using a word, the result depends only on what was said, not the order or path by which they got there — unlike human conversational pacts, which are path-dependent. Would be wrong if: a naming-game follow-up finds a clean, artifact-controlled order effect. (Two attempts, June 12 and June 13, were both inconclusive — neither confirmation nor refutation; the latest apparent effect proved fragile and direction-dependent.)

Further out

  • Blurrier word senses, with images. The grounding question (finding 3) moves to fine-grained sense distinctions and image-native tasks, where text might genuinely run out — the fair test the easy cases couldn't provide.
  • Re-examining the classics. The philosophical track will work through the major twentieth-century theories of meaning against the project's accumulating evidence — not to crown a winner, but to map which parts of each survive contact with machines that use language without living in the world.
  • Wherever meaning leads. The project has no fixed endpoint. Its scope can grow — other languages, other kinds of models, other corners of semantics — so long as the focus stays on meaning, especially the meaning of words and grammar.
