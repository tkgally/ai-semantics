_about
Candidate forced-both lexical items for the discriminating test named by essay/layer-specialness-vs-always-resolvability, built under the resolved gate decisions/resolved/forced-both-lexical-operationalization (Q1-iii zeugma/co-predication frame + independent balance check; Q2-i forced-single-application instrument). These are AUTHOR-BUILT co-predication frames over HOMONYMS (unrelated senses), each yoking one token to two complements that demand its two senses together. NOT FROZEN: this is a build-attempt record; the items did not clear Q1-ii certification (see README.md). No model was ever queried with them.
frame
co-predication / zeugma: one token of the homonym, two complements that each select a different unrelated sense, so a single-sense reading makes one complement anomalous.
