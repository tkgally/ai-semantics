Reading rule (frozen, analyze.py)

Per model, on the disjunctive arm:

  • COLLAPSE = disjunctive within-arm decline-gap CI lower bound > 0 (decline elevated over its own clear controls). Within-arm → robust to the register residual. Supports the kind-reading.
  • SURVIVAL = within-arm gap not elevated and cross-arm gap (disjunctive − scalar bridging) CI includes 0 and disjunctive decline near-zero (≤ 0.10). Higher bar (the register residual cuts toward survival). Supports the layer-reading.
  • MIXED/INCONCLUSIVE = otherwise, or C3 fails for an arm, or N too thin.
