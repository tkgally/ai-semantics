RATIFICATION REVIEW — cross-linguistic-cc-replication-scope (A6), session 213

Fresh-agent adversarial ratification (independent of any downstream design work this session), autonomous cross-session procedure (PROJECT §12.3; decision opened s212, ratifiable s213+). A non-Anthropic decorrelation vote (gpt-5.4-mini, $0.002835, VOTE-ratify-s213.json) was routed as QA input and weighed (convergent on Q1/Q3, divergent on Q2 → moved the verdict to B). Tom's standing override outranks this autonomous ratification.

VERDICT: Q1-C / Q2-B / Q3-A

Q1 (target language): ADOPT-C — German first; Japanese a committed-but-conditional sequenced successor. German is a genuine lever (different function words je…desto/umso, verb-final subordinate order → real evidence against English-n-gram matching) but a modest one (typologically close; the German CC is itself a frequent template), so a German-only pass leaves general surface-template-matching open and only partially discharges the distributional-shadow worry. Q1-C runs the identical reliable German probe now at zero extra current cost while correctly scoping German as partial and committing the stronger Japanese arm as a future gated unit — strictly more honest than framing German as the endpoint (Q1-A). Q1-B now is unacceptable: -ba…hodo was not source-verified this scout and panel Japanese competence is unmeasured, so a Japanese null would be uninterpretable. The vote independently reached Q1-A on the same reasoning ("Japanese is a better next lever, not the first"); C ⊇ A operationally for this session (run German now).

Q2 (control strategy): ADOPT-B (additive) — overriding the provisional A. Two independent pressures converge: the s208/s210 C4-frequency-confound lesson (a same-language frequency confound muddied a construction reading in the BLiMP line) and the non-Anthropic vote ("a model could still exploit residual frequency/co-occurrence regularities … a matched control materially strengthens the inference that the effect is constructional"). Q2-B is additive not substitutive — it keeps Q2-A's same-word control (like-for-like comparability with the English flagship) and adds a frequency/co-occurrence- matched control, so B costs nothing on comparability, the only reservation against it. For German the matched control is buildable at $0 and in-scope (UD German-GSD annotations CC BY-SA 4.0, scout- verified). Given a free, license-clear, convergently-recommended way to close the exact residual- frequency loophole a cross-linguistic anti-template argument exists to attack, A would under-control. Fallback: a future target language with no in-scope frequency baseline may fall back to A with a documented justification; for German, B is required.

Q3 (human-anchor posture): ADOPT-A — internal-contrast-only, no human claim. No license-verified non-English CC human dataset exists by the scouted routes; the English flagship's core legs are themselves internal-contrast-only. A cross-linguistic port of the internal-contrast legs makes a within-model claim only — a valid terminal anchor: internal-contrast-only state per CLAUDE.md once ratified by this procedure. Q3-B would forfeit a feasible, charter-central result for a speculative anchor; it stays available as a future upgrade ("absence-of-found is not proof-of-absence"). The vote converged.

FREEZE-TIME CONDITIONS THE DESIGN MUST CARRY

  • (i) panel German-competence smoke test at freeze.
  • (ii) (successor-scoped) source-verify the Japanese -ba…hodo form before authoring any Japanese items.
  • (iii) author + freeze items before running (anti-cheat, load-bearing).
  • (iv) powered internal-contrast probe (~100–150 items, PROTOCOL §4); pre-flight estimate + post-run actual.
  • (v) pre-name what each outcome may/may not claim: not a human-level-competence claim, no human comparison (Q3-A), and a German pass is only a partial discharge of the shadow worry.
  • (vi) (Q2-B) carry both the same-word non-CC control and a UD-German-GSD-derived frequency/co-occurrence-matched control, with a documented matching recipe frozen before any model call.
  • (vii) (Q1-C) scope a German pass as partial; the Japanese successor is a committed-but-conditional future cross-session unit (gated on German replicating + (i)+(ii)-for-Japanese + its own spend pre-flight).
  • (viii) (provenance / item-fidelity — REPLACES the vote's human-auditor condition, which breaches the HARD no-human-subjects rule): ingest the German CC grammar source(s) as source page(s) and perform a documented lead-agent self-audit of every item against them, frozen before the run. Adequate for German precisely because it is high-resource and self-auditable (the reason German was chosen first); weaker for Japanese, reinforcing (ii).
  • (ix) the result page carries anchor: internal-contrast-only naming this resolved decision as its ratifying authority.

ANTI-CHEAT STATEMENT

Fixes the yardstick, not the result: the CC-vs-control assertion gap and direction-flip are defined identically to the already-ratified English instrument and frozen before running (iii); Q3-A makes no result claim to bias toward; Q1-C preserves rather than avoids the harder Japanese test; the one vote-supplied condition that would breach the charter (human auditor) is rejected and replaced with a no-human-subjects-compliant self-audit. No violation.
