Pipeline

  1. prep.py — derived human_cell_means.csv (204 cells) + human_class_means.csv (28 class-cells) from all 3,600 non-filler Exp-2 ratings; froze stimuli.json (seed 20260612): 408 anchored / 102 robustness / 60 held-out / 24 Tier-0. No model calls.
  2. Independent pre-run critic → 3 blockers + 8 should-fixes, all applied pre-run (commit b811065), including the noun-class guard (a noun-class-marginal responder with zero adjective information would have passed the original thresholds at ρ=0.466/0.613).
  3. probe.py — 1,782 calls (temp 0; A/B max_tokens 64, C 512 + reasoning effort minimal).
  4. analyze.py — implements the frozen map exactly → results.json.
  5. Independent post-run verifier — recomputed every gate from raw with independent code.
