experiments · 2110 files in total

Experiment designs, run records, and data. This is where the actual stimuli (the sentences shown to the models) and the raw model responses live.

Folders

  • data/ (27 files) Datasets and corpora prepared for experiments (constructed batteries and samples of external human-labeled resources).
  • designs/ (186 files) Frozen experiment designs — the pre-registered plan that turns a conjecture into a probe.
  • lib/ (2 files) Shared experiment library code (the OpenRouter client the probes call through).
  • notes/ (3 files) Feasibility notes and literature scoping written alongside the experiments.
  • runs/ (1892 files) One folder per experimental run: the stimuli, the analysis code, the results, and the raw model calls.
