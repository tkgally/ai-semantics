Decision: re-design the cancel arm of the monotonicity-generalization battery

Why this exists

decisions/resolved/constructional-monotonicity-generalization-operationalization (ratified s134, ADOPT-WITH-MODS) fixed the matched-difficulty add-vs-cancel battery's yardstick, with CANCEL = the for-durative aspectual coercion of a semelfactive, indicator affirm-"happened only once". MOD-3 flagged that default as contestable and prescribed the failure path: a B2 NO-GO re-pairs the cancel coercion before any spend.

Session 135 built and froze the battery and ran the B2 gate. The gate is a NO-GO (note/monotonicity-generalization-b2-nogo-v1): the add arm passes cleanly (construction 1.00 / control 0.00, 3/3 models), but the cancel default affirm ("only once") is 0.00 in 3/3 models under both NLI and forced-choice — uniform NLI label 1 (neutral). The failure is structural, not verb-selection: single-occurrence is a defeasible Gricean quantity implicature, not a lexical entailment, so a strict-NLI default can never reach the ≥ 0.80 matched ceiling for any semelfactive verb.

Therefore a within-frame re-pair (different semelfactive verbs) is futile, and the only real fix is a re-designed cancel coercion whose default is a genuine (near-)entailment at ceiling on the bare frame — which changes the yardstick and so is a cross-session decision (charter §12.3), surfaced here, ratifiable no earlier than the next session.

What stays fixed (NOT re-opened): the ADD arm (resultative, verified pool), the B2 gate machinery, and every frozen threshold (B2 0.80/0.40; asymmetry 20 pp; instrument-fragility 10 pp; cue ≥ 0.70). Only the choice of cancel coercion is in question.

The named failure mode this gate guards (unchanged)

Tuning the construction pair / difficulty bar until the asymmetry appears. The guard is unchanged: freeze-and-hash + independent pre-run critic + live falsify arms. A re-designed cancel arm must itself clear the B2 default-at-ceiling gate before any spend — do not repeat the s135 trap by assuming a new default is at ceiling; measure it.

Options for the re-designed cancel coercion

Each must (i) be a genuinely new cancel direction (beyond caused-motion / way / conative); (ii) have a bare-frame default that is a (near-)entailment plausibly at the ≥ 0.80 B2 ceiling under strict NLI; (iii) be cancellable by an in-sentence construction; (iv) remain internal-contrast-only unless a human anchor is fetched.

  • Option C1 — Telic-completion (accomplishment) cancellation. Default: a simple-past accomplishment entails completion ("Maria painted the house" → "Maria finished painting the house"); the for-durative / progressive-leaning atelic frame cancels it ("Maria painted the house for two hours" need not entail completion — the imperfective-paradox cancellation). Trade-off: keeps the aspectual-coercion spirit of the ratified arm (so the pair stays a natural cousin of what was ratified) and completion is a far stronger default than single-occurrence — but completion is itself somewhat defeasible under strict NLI ("painted the house but didn't finish" is marginal, not contradictory), so its default-at-ceiling is unverified and must clear B2 exactly as MOD-3 requires. This is the provisional default only if a quick frozen B2 calibration confirms the completion default ≥ 0.80; otherwise it fails the same way "only once" did.
  • Option C2 — Privative-modifier cancellation. Default: a bare head noun entails its category ("a gun" → "is a weapon"); a privative modifier cancels it ("a fake gun" → not a weapon; "a toy gun", "a former senator"). Trade-off: the cleanest, most ceiling-likely lexical default and the sharpest cancellation — but it is adjectival/nominal, not a verbal argument-structure construction, so it drifts from the conjecture's constructional scope (the asymmetry was abstracted over verbal constructions); using it would broaden the conjecture's claim, which should be deliberate, not incidental.
  • Option C3 — Same for-durative frame, a stronger (entailment) indicator. Keep the semelfactive + for-durative sentences but cancel an inference that is an entailment of the bare verb rather than the implicature "only once" — e.g. the punctuality/brevity of the event, or a definite single bounded result. Trade-off: minimal change to the ratified frame, but it is unclear any entailment of a bare semelfactive is what the for-durative cancels (the for-durative cancels exactly the single-occurrence implicature); likely to floor again. Lowest-confidence option.
  • Option C5 (RULED OUT, recorded) — within-frame verb re-pair. Pick different semelfactive verbs under the unchanged "only once" indicator. Ruled out by s135 evidence: the floor is structural (uniform 0.00 across flash/jump/cough/nod, 3/3 models, both instruments), so no semelfactive verb can reach the B2 ceiling. Named only to record it has been tried-by-implication and rejected.

Provisional default (NOT ratifiable until a later session)

Option C1 (telic-completion), conditional on its own B2 calibration. It preserves the aspectual-coercion character of the ratified arm and targets a much stronger default than single-occurrence, while staying a genuinely new pair and internal-contrast-only. The condition is binding: a future session must run a frozen B2 calibration of the completion default first and only proceed if it clears ≥ 0.80 in ≥ 2/3 models — if it floors like "only once" did, fall back to C2 (accepting the deliberate scope-broadening) or record the matched-difficulty cancel arm as un-instrumentable on the project's strict-NLI panel (an honest null about the method, parallel to the grounding-magnitude un-instrumentability already on record).

What the later ratifying reviewer should weigh

  1. Is C1's completion default likely to clear B2, or will simple-past accomplishments also read as neutral under strict NLI (the same trap one rung up)? Should the ratification be conditional on a fresh completion-default calibration, as the provisional default proposes?
  2. Is broadening to C2 (privative modifiers) an acceptable widening of the conjecture's constructional scope, or does it change what the conjecture is about?
  3. Should the project accept a principled-limit closure — "the matched-difficulty cancel arm is not instrumentable at ceiling on a strict-NLI panel without a lexical-entailment default" — as a first-class outcome if C1 and C2 both prove unsatisfactory, rather than chasing further coercions?
  4. Does the s135 finding (the cancel default is too weak to reach ceiling) bear on the conjecture's own framing — i.e. is "cancellation is hard" partly that the cancelled inferences the project reaches for are implicatures the models never held as entailments?

Anti-cheat (binding before any spend)

The re-designed cancel arm is frozen + sha256-hashed before the first probe; its default-at-ceiling is certified by the B2 gate before the asymmetry is read; an independent pre-run critic GOes against the frozen set; the falsify arms (symmetric, reversal) stay live. Ratifying this decision fixes the yardstick, never the result.

Ratification — session 136 (2026-06-28): ADOPT C1 (conditional) → C2 fallback → closure, MODS M1–M3

Per PROTOCOL.md §2, an independent fresh-agent adversarial review (not the s135 session that opened this decision, and not this session's essay-writing units) read this decision in full, its options, the provisional default, the two downstream contingent artifacts (conjecture/constructional-monotonicity-asymmetry, note/monotonicity-generalization-b2-nogo-v1), the parent ratified yardstick, the cc-v2 paradigm, the s133 add-arm headroom calibration, and the frozen battery, and returned ADOPT C1 (telic-completion) CONDITIONAL on its own B2 calibration, with C2 (privative) as the pre-authorized fallback and a principled-limit closure only if both fail — WITH MODS M1–M3.

Why C1 first (the crux, weigh-question 1). Completion is a materially stronger default than single-occurrence and the difference is principled, not wishful: single-occurrence is a Gricean quantity implicature (a semelfactive is a repeatable point-event), whereas simple-past accomplishment culmination is closer to an aspectual entailment encoded in the predicate's event structure. The resultative add arm — which licenses a result-state/completion-type entailment — already clears ceiling cleanly (construction affirm 1.00, 3/3 models), showing these models do treat result-state content as a strict entailment when structurally present, the opposite posture from how they treated the "only once" implicature. But completion is itself somewhat defeasible under strict NLI ("painted the house but didn't finish" is marginal, not contradictory; the imperfective paradox lives next door), so its default-at-ceiling is unverified and must be measured — exactly the s135 lesson. The conditional structure neutralizes the spend risk: a completion-default B2 calibration is a ~12-item, ~$0.02-class probe, and a NO-GO spends nothing on the main battery and triggers the C2 fallback. An unconditional C1 adoption would repeat the s135 error and is explicitly rejected.

Why C2 is fallback, not primary (weigh-question 2). C2's privative cancellation ("a gun" ⊨ "a weapon"; "a fake gun" cancels it) is the cleanest, most ceiling-likely lexical default — but it is nominal/adjectival, not a verbal argument-structure construction, so it broadens the conjecture's constructional scope, which must be deliberate, not incidental (MOD-2). C1 preserves the aspectual-coercion character of the ratified arm and keeps the conjecture's scope unchanged.

Why a principled-limit closure is first-class (weigh-question 3). If both C1 and C2 fail their B2 gates, the honest terminal outcome is that the matched-difficulty cancel arm is not instrumentable at ceiling on the project's strict-NLI panel without a lexical-entailment default the models hold categorically — a null about the instrument/method, parallel to the on-record grounding-magnitude un-instrumentability. It confers no support on the conjecture and leaves it proposed. It is the third stop (C2 is genuinely ceiling-likely), and is triggered only by measured B2 NO-GOs (M3).

Weigh-question 4 (does s135 bear on the conjecture's framing). Yes, and it reinforces the gate-before-asymmetry discipline rather than destabilizing the decision: the whole point of the B2 default-at-ceiling gate is to guarantee a real, model-held entailment to suppress, structurally excluding the "the inference was never an entailment" confound from any asymmetry verdict. The companion philosophical essay essay/nothing-to-cancel (written this session) generalizes exactly this point; it bears on the ratification only as vindication of the gate's necessity and does not alter the verdict.

The frozen instruction the build session inherits (no latitude to retune)

  1. STEP 1 — completion-default B2 calibration FIRST. Freeze + sha256-hash a calibration set for a telic-completion (accomplishment) cancel arm before any API call: default = simple-past accomplishment ("Maria painted the house"), bare-frame default hypothesis = completion ("Maria finished/completed …" — pick one phrasing, freeze it); cancelling construction (held OUT of calib) = the for-durative/atelic frame. Use ≥6–8 genuine accomplishment verbs whose bare simple-past plausibly entails culmination, disjoint from the resultative ADD pool (do NOT reuse beat/boil/cut/hammer/kick/paint/push/scrub/squeeze/wipe — note paint is in the add pool, so the accomplishment item must not be a resultative; use e.g. build, write, read [a book], bake [a cake], knit [a sweater], mow [the lawn]). Run NLI-primary + FC, temp 0, the ratified panel. GATE (unchanged, not relaxable): completion default affirm ≥ 0.80 in ≥2/3 models (strict NLI, label 0 = entailment). - GO → STEP 2 with the telic-completion cancel arm; conjecture scope unchanged. - NO-GO → do NOT relax the bar or re-pair completion verbs within the frame (a NO-GO is structural, like s135); record a verified C1-NO-GO result (a weigh-question-4-bearing finding — completion, too, read as defeasible-not-categorical under strict NLI), then go to STEP 1b.
  2. STEP 1b — privative-default B2 calibration (C2, ONLY on C1 NO-GO). First apply M2: amend conjecture/constructional-monotonicity-asymmetry in writing (broaden scope to include a nominal/adjectival privative cancel arm; verbal-only abstraction relaxed; add-verbal/cancel-nominal mismatch a live caveat). Then freeze + gate a privative calibration: default = category membership ("a gun" ⊨ "a weapon"), cancel = privative modifier ("a fake/toy gun" ⊭ weapon), ≥6–8 pairs; same gate (category-membership default affirm ≥ 0.80 in ≥2/3 models). GO → STEP 2 (privative arm + amended conjecture); NO-GO → STEP 3.
  3. STEP 2 — build, gate, critic, run (only on a B2 GO). Reuse the s135 frozen ADD arm verbatim (resultative; items sha 80bd4b60b55a3e60). Build the matched cancel arm (C1 or C2) in the same conflicting-cue paradigm (default/construction/cue). Freeze + sha256-hash the full set; re-run the full B2 gate on the main-item calibration subset (add headroom: construction ≥ 0.80 AND control ≤ 0.40; cancel default ≥ 0.80; all ≥2/3 models); obtain an independent fresh-agent pre-run critic GO; then run the full battery, read by the frozen s134 thresholds, untouched (asymmetry 20 pp; instrument-fragility ≥ 10 pp; cue ≥ 0.70). Post-run, an independent fresh agent recomputes every number. The result is internal-contrast-only. M1: the result page must state that the B2 pass is what licenses reading any suppression failure as defeasance.
  4. STEP 3 — principled-limit closure (ONLY if both C1 and C2 NO-GO). Record, as a first-class result, that the matched-difficulty cancel arm is un-instrumentable at ceiling on the strict-NLI panel without a categorically-held lexical-entailment default; confers no support on the conjecture, which stays proposed. Not reachable if either B2 passed (M3).

Spend note: each B2 calibration is a ~12-item, ~$0.02-class probe (s135's was $0.01720); STEP 1 and 1b together are well under $0.10 — budget is not a constraint on running C1's calibration first.

Anti-cheat affirmed by the review. This fixes the yardstick, never the result: it blesses which cancel coercion to try next and the unchanged requirement that it clear its own B2 default-at-ceiling gate before any asymmetry is read — not whether the asymmetry confirms. The binding condition on every option is a measured default at ceiling, with NO-GO → fall back (never relax the bar) — anti- confirmation machinery, not pro-confirmation. The frozen s134 thresholds (B2 0.80/0.40; asymmetry 20 pp; fragility 10 pp; cue 0.70) are untouched and unused here; the falsify arms (symmetric, reversal) stay live; the conjecture stays proposed; no battery is run and no threshold re-tuned in this ratifying session.
